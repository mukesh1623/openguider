Welcome to OpenGuider

Before first use, review these setup and security notes.

  1. Open Settings and add your AI provider API key. API keys are obtained directly from your selected provider dashboard (OpenAI, Anthropic, Google, Groq, OpenRouter, Ollama, etc.).
  2. Required permissions are microphone access (push-to-talk) and optional screen capture guidance. Installer metadata can document these permissions; runtime prompts are requested only when needed.
  3. Your API keys are stored locally using secure OS storage when available (with encrypted fallback). OpenGuider only sends HTTPS requests to the AI providers you configure.
