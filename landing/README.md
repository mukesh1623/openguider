# Landing Page Notes

## Download URLs

Landing download buttons are linked to the latest GitHub release assets:

- `OpenGuider-windows-latest.exe`
- `OpenGuider-macos-latest.zip`
- `OpenGuider-linux-latest.AppImage`

These alias files are generated by `.github/workflows/release-build.yml`.

## GitHub Pages deployment

The landing page is deployed by `.github/workflows/deploy-landing.yml`.
It publishes:

- `landing/index.html`
- `landing/landing.css`
- `landing/landing.js`
- `tutorial.gif`
- eye animation files from `renderer/assets/`

## Eye animation assets

This landing page blinks with these files in order:

1. `logo.png`
2. `half-opened.png`
3. `full-closed.png`
4. `half-opened.png`
5. `logo.png`

Preferred location:

- `landing/assets/`

Fallback location (already present in this repo):

- `renderer/assets/`

If you deploy only the `landing/` directory somewhere else, copy these 3 image files into `landing/assets/`.
