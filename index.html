    Security

    Open-source and local-first by design.

    How OpenGuider Handles Data

    • Open-source codebase, auditable by anyone.
    • API keys are kept locally on your machine.
    • No hidden cloud relay between you and providers.

    Where Requests Go

    • Only to providers you configure (AI, STT, TTS).
    • HTTPS/TLS for data in transit.
    • No unrelated third-party telemetry endpoint by default.
